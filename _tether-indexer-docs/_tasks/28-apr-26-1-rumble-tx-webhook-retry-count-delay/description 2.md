# Description

**Environment:** App version: iOS: v3.0.1 339. Used device: iPhone 17 Pro Max, iOS version 26.2.

## Description

When creating a new user or creating a new channel and starting to follow it, on the web the tip button doesn't appear and in the app Send Tip displays as inactive. After relogin the button appears.

Note: I am not sure if it is from our side or Rumble side, but as far as I know it is important, I am creating this issue.

## Steps to Reproduce

1. On the web create a new user or channel
2. Start following it

**Actual Result:** Tip button doesn't appear.

3. Check on the app

**Actual Result:** Send Tip button is inactive.

## Expected Result

Tip button should appear and Send Tip button should be active.
